# Security & Privacy Policy for Monetra

## Privacy-First & Offline Philosophy

Monetra is strictly an **offline-first, privacy-first personal finance application**.

- **Zero Telemetry**: Monetra collects zero analytics, zero crash logs, zero usage metrics, and zero telemetry.
- **Zero Cloud Authentication**: Monetra does not communicate with external servers, cloud databases, or authentication providers.
- **Local Storage**: All financial records, accounts, budgets, and settings remain stored on your local device inside an encrypted SQLite/Drift database vault.

---

## Reporting Vulnerabilities

If you discover a security vulnerability or privacy bug within Monetra, please report it privately to the maintainers rather than opening a public issue.

### Contact Information
- **Security Lead**: `security@monetra.app`
- **Response SLA**: Initial triage response within 48 hours.
